# discord-giveaway-bot
 A discord giveaway manager bot

# Persyaratan minimum untuk menjalankan bot
- Nodejs versi 16.6 keatas
- Discord bot token

# Cara menjalan kan bot
- Ganti `.env.example` ke .env
- Isi semua value yang ada
- jalankan `npm start`

# Larangan keras
